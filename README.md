# ポケコン PC-1245/50/51/55用ゲーム 『ポケコン電卓　numbers』

### 「俺の恋人はゲームだぜ〜！！」

## 動作機種
SHARP PC-1245/50/51/55系<br>
※プログラムの実行には、Fan_PC-1251様作『`PC-インタープリタ`』（工学社PiO 86年8月号掲載）が必要となります。

## 開発言語
PC-インタープリタ・マシン語

| `README.md`       | マニュアルファイル                   |

## セットアップ

本体へ、以下の順でセットアップします。

現在、プログラム音声ファイルは公開していません。<br>
よって、マシン語データファイル`numbers.hex`とプログラムファイル`numbers.bas`を、何らかの方法でプログラム音声ファイルに変換して転送するか、プログラムリストを手入力する必要があります。<br>

### 補足
`numbers_org.bas`は、`numbers.bas`にコメントの追加と、命令文等の省略を無くしたものとなります。<br>
どちらも、処理内容に違いはありません。<br>
（プログラムの可読性を高めたものです。）<br>

PC-1245の場合、メモリの都合上、途中で入らなくなります。<br>
PC-1251/55であれば入力できますが、`コメント 'の行`は削除してください。<br>

セットアップ後にCLEARを実行すると、データが消去されるため、使用しないでください。<br>

## 起動方法

`DEFキー`を押し、`=キー`を押してください。<br>

## ゲームのジャンル

数字ベーダー撃退アクション<br>

## ストーリー

手持ちのポケコンが、突如「誤作動」を起こした。<br>

画面の右端に現れる、見慣れない“数字”。<br>
それはただの表示エラーではなかった。<br>

それらは・・<br>
知性を持った数値生命体《数字ベーダー》だった。<br>

## ゲームのルール

右端から`数字ベーダー`が出現し、プレイヤーのいる左へと迫ってきます。<br>

プレイヤーの自機は、`0〜9`および`ENTER`を押下することにより、最も左にいる数字ベーダーを撃退します。<br>

数字ベーダーが左端まで到達すると、侵略されてミスとなります。<br>
３回侵略されると、ゲームオーバーとなります。<br>

## 操作方法

### 《ゲーム》

| キー    | 役割                                 |
|---------|--------------------------------------|
| `0〜9`  | 数字ベーダーと同じ数字で撃退します。 |
| `ENTER` | UFOを撃退します。                    |

## キャラクター紹介

### 《数字ベーダー》

知性を持った数値生命体です。<br>

### 《UFO》

時折出現する、数値生命体の“指揮官クラス”です。<br>
撃退すると、ボーナスが入ります。<br>

## スコア

| 行動               | 得点     |
|--------------------|----------|
| 数字ベーダーを撃退 | １〜６点<br>撃退した桁に応じた得点（右へいくほど得点が高い） |
| UFOを撃退          | １００点 |

## 終了方法

本体の`BRKキー`を押すか、ゲームオーバーで終了します。

## 変数表

| 変数     | 内容                                                 |
|----------|------------------------------------------------------|
| n (0〜6) | 表示桁にいる数字ベーダーの番号<br>UFOは '0A'         |
| A〜E     | ワーク用<br>（主にドットキャラクターを表示する際に使用） |
| G        | 数字ベーダーの撃退回数                               |
| L        | ドットキャラクター表示サブルーチンの引数：横表示位置 |
| M        | 最大表示桁<br>６桁から開始し、徐々に減る（最小２桁） |
| R        | サブルーチン実行後の戻り行                           |
| S        | スクロールカウンタ                                   |
| T        | 得点下位２桁<br>（０〜９９）                         |
| U        | 得点上位３桁<br>（１００〜２５５００）               |
| W        | ワーク用                                             |
| Z        | 自機の残数                                           |
